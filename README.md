## Getting Started

1. Open `index.html` in your web browser
2. Customize the content with your information:
   - Update your name and title in the hero section
   - Add your about information
   - Update project cards with your projects
   - Modify contact information
   - Update social media links

## Customization

### Colors
Edit the CSS variables in `assets/css/style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    --accent-color: #f59e0b;
    /* ... more variables */
}
```

### Content
- Update text content in `index.html`
- Modify project cards in the Projects section
- Change statistics in the About section
- Update contact information

### Fonts
The website uses Google Fonts. To change fonts:
1. Update the Google Fonts link in `index.html`
2. Update font variables in CSS:
```css
--font-primary: 'Your Font', sans-serif;
--font-display: 'Your Display Font', serif;
```
